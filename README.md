# 🌑 Nyx

**Nyx** is a **CLI-based role-playing game** written in **x86-64 NASM Assembly**.
Step into the shadows and embrace the darkness — the terminal is your world.

## 🚀 How to play

### 🛠️ Build Nyx
```bash
make all
```

### 🎮 Run the game
```bash
./nyx
```
Let the story unfold in your terminal...
